Settings
--------

You can set defaults for ``django-propeller`` in your settings file. Refer to :doc:`settings` for more information.


Demo application
----------------

The demo application provides a number of useful examples.

clone the repo::

    $ git clone https://github.com/tfroehlich82/django-propeller.git

cd into the cloned directory::

    $ cd django-propeller

run the testserver::

    $ python manage.py runserver

open your browser and browse to::

    http://127.0.0.1:8000
